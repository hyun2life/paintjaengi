# Paintjangi Cloudflare Pages Deploy

이 폴더는 `Cloudflare Pages`에 바로 업로드할 수 있게 정리한 배포용 폴더입니다.

## 포함 파일
- `index.html`
- `styles.css`
- `CI.jpg`

## 업로드 방법
1. Cloudflare 로그인
2. `Workers & Pages` 이동
3. `Create application` 선택
4. `Pages` 선택
5. `Upload assets` 또는 `Drag and drop your files` 선택
6. 이 폴더 전체를 업로드

## 배포 후 확인
- 기본 주소는 `프로젝트명.pages.dev`
- 연락처 문구는 아직 플레이스홀더 상태이므로 나중에 실제 정보로 교체 필요

## 수정 위치
- 문구 수정: `index.html`
- 디자인 수정: `styles.css`
- 로고/CI 교체: `CI.jpg`
